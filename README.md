# TURIVIUS - LIVROS
## How to set up
- Clone the project:
  > `git clone https://github.com/vinesnts/turivius-livros.git`
- Setup python enviroment:
  > `python3 -m venv env`
- Enter the new enviroment:
  > `source env/bin/activate`
- Install requirements.txt:
  > `pip install -r requirements.txt`
- Run project:
  > `gunicorn main --reload`
